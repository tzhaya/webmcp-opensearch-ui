農林水産関係試験研究機関総合目録（ALIS WebOPAC・dcndl）

ALIS WebOPAC は農林水産研究情報総合センターが提供する農林水産省が所管する試験研究機関・国立研究開発法人図書館の蔵書統合目録です。本フォームはOpenSearch APIでの検索により、DCNDL形式での返戻結果を利用（format=dcndl 固定）し、著者典拠 URI / 読み / NDC 分類記号 URI / CiNii NCID への rdfs:seeAlso など、書誌セマンティクスを保持した形で結果を表示します。 WebMCP ツール名は searchAffrcOpac。

CiNii の appid を設定する（任意・端末内に保存）

CiNii Research OpenSearch v2 のドキュメント上 appid は必須とされています。 本デモは未設定でも試行しますが、サーバ側で拒否される場合は CiNii のサイト で取得した appid をここに登録してください（localStorage にのみ保存されます）。

統制語彙（NDLSH / AGROVOC）の既定参照先を設定する

suggestSearchTerms ツールが vocabularies 引数を省略された場合に 参照する語彙の既定値です（localStorage にのみ保存）。 vocab.html で挙動を直接確認できます。

ALIS WebOPAC プロキシ URL を設定する（CORS 回避用・端末内に保存）

ALIS WebOPAC (library.affrc.go.jp) は CORS を許可していないため、 ブラウザから直接 searchAffrcOpac を呼び出すとブロックされます。そのため jc-opensearch-client の Cloudflare Worker を ALIS WebOPAC 対応に拡張したプロキシ経由で叩きます。 既定で動作するプロキシ URL があらかじめ設定されています。別のプロキシを使う場合のみ上書きしてください （localStorage にのみ保存）。 上流の AFFRC 側が国内向けに制限されているため、国外からは利用できません。

分類記号（NDC/NDLC）レコメンドの有効/無効

searchPaper の戻り値ヒント (expansionHint) に 「suggestClassificationCodes ツールで NDLSH 件名標目から NDC/NDLC を取り出し 書籍検索の category パラメータに OR で渡す」案内を含めるかどうか。 無効化してもツール自体は登録・呼び出し可能です。
